=============================== 助记词钱包（HD钱包）
//======================================================================== 助记词钱包（HD钱包）

一个助记词可以派生多个私钥，从而派生多个不同链的钱包地址（通过不同的路径规则），这叫 HD钱包（分层确定性钱包）。
		//相当于我以同一个原料生成不同的随机数
		//一个“原料”（助记词） + 不同的“路径规则” ⇒ 生成多个“私钥” ⇒ 多个地址

		//HD钱包（分层确定性钱包）
		//助记词 相当于“种子”；
		//	不同的路径（如 m/44'/60'/0'/0/0）像“配方”；
		//	得到不同的私钥、不同的区块链地址；
		//	所有私钥都能从这个助记词中重新“派生”出来。
		//所以你记住助记词，就相当于掌握了整个钱包家族的根本。

所以你记住助记词，就相当于掌握了整个钱包家族的根本。

//======================================================================== 助记词钱（HD钱包的）与 私钥 的转换
//======================================================================== 助记词钱（HD钱包的）与 私钥 的转换


助记词 → 私钥 ✅（常见，标准流程）
	通过 BIP-39 + BIP-32 等算法，从助记词 派生出私钥

私钥 → 助记词 ❌（不常见，几乎不做）
	私钥本身是最终结果，不能直接反推出助记词，因为它是通过哈希+派生算法生成的，不是可逆的。

助记词可以生成私钥，但 私钥不能还原为助记词。
//助记词 → 私钥，助记词相当于生成非对称密钥对时的随机数种子，用于推导出私钥。

 
//========================================================================  助记词钱包 与 私钥钱包
//========================================================================  助记词钱包 与 私钥钱包

助记词钱包  = 可派生多个账户的钱包结构；
			账户01
			账户02

私钥钱包 = 单一地址的钱包；

//它们的资产是彼此独立、互不相通的。

//======================================================================== 举个例子图示说明：
//======================================================================== 举个例子图示说明：

（1）钱包 A（由助记词生成）：
		├─ 账户 01 → 地址：0xabc123… → $2.81
		├─ 账户 02 → 地址：0xdef456… → $0
		├─ 账户 03 → 地址：0xaaa999… → $0

（2）私有钱包
	私钥钱包（手动导入）：
	├─ 地址：0x999999… → $0.19

/**
	它们就像是：
		钱包 A = 你拥有一串万能钥匙（助记词），可以打开多个抽屉（账户 01、02…）；

		私钥钱包 = 你单独拿了一把钥匙（私钥）开一个独立抽屉；
*/

钱是分开存放的；

你转进钱包 A 的账户 01，不会自动出现在私钥钱包中；
除非你主动转账，否则它们互不影响。
//======================================================================== 不管是 助记词钱包，还是私钥钱包，它们的私钥格式相同
//======================================================================== 不管是 助记词钱包，还是私钥钱包，它们的私钥格式相同

私钥格式（以以太坊为例）
	通常是 64 个十六进制字符（256 位），比如：
	0x4c0883a6910395bae1234567890abcdef1234567890abcdef1234567890abcdef

地址格式（以以太坊为例）
	0x 开头 + 40 个十六进制字符（共 20 字节）
	例如：0x742d35Cc6634C0532925a3b844Bc454e4438f44e



无论私钥是怎么来的（助记词派生 or 直接生成），一旦生成，它的格式与功能都是一致的，
只不过地址可能属于不同链（比如以太坊、BTC、BSC 等）——这取决于你如何使用该私钥（即使用哪条链的规则去派生地址）
