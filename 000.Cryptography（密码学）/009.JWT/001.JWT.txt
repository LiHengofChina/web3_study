


JWT（JSON Web Token）是一种标准，


用于在各方之间 "安全地传输信息"。
它可以 "使用签名（数字签名）" 或 "消息认证码（MAC）" 来确保数据的 "完整性和真实性"。



//=================================================================== JWT 简介：
//=================================================================== JWT 简介：



JWT 是一种 "开放的标准（RFC 7519 ）"， 它定义了一种 "紧凑的" 、"URL安全的" 方式，
用于在各方之间 "传输声明"。

	/***
		重点来了： 
				是 "传输声明" ，并不是加密它，它被截取就能伪造。
	*/
