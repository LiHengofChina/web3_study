keytool 是一种类似于 OpenSSL 的工具，专门用于管理和生成加密密钥、证书和  "密钥库"。

	 它通常与 Java 环境中的密钥和证书管理相关
	 特别是在创建 Java 应用程序中的 SSL/TLS 安全通信时

	（1）keytool 可以生成密钥对、
	（2）导入和导出证书，
	（3）并创建和管理 Java 的密钥库（keystore）和信任库（truststore）。
