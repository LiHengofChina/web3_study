//================================================（1）与 PEM 格式：

PEM 通常单独保存证书（cert.pem）和私钥（private_key.pem），
且每个文件是纯文本（Base64 编码）。
PKCS12 则将私钥、证书链等打包在同一个文件中，整个文件经过加密保护。

//================================================（2）与 JKS（Java Keystore）格式：
//================================================（2）与 JKS（Java Keystore）格式：

JKS 是 Java 应用程序中常用的密钥库格式，主要用于 Java 密钥库。
PKCS12 更通用，并且现代 Java 应用也支持使用 PKCS12 格式的密钥库。
